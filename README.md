Это репозиторий-шаблон для проекта Яндекс Практикума спринта №1.

Проект разделен на 3 этапа:
1. Сбор данных
2. Очистка данных
3. Обучение модели

Весь код, обеспечивающий решение 1 и 2 этапов, сохраните в папке этого репозитория под названием `part1_airlfow` <br/>
